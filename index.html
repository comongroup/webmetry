WEBMETRY

Webmetry is a tool that will help you debug some of your site's styles while you're developing it.

To see it in action, press Shift+W to toggle it.

Still in development!

We are actively developing both Webmetry and this website, and we know there are many things still missing.

Feel free to open an issue or a pull request if you want to contribute.
